### tao-core

后端开发中的一些常用组件与抽象定义：

- 事件总线

- 通用查询组件（CommonQuery）

- 字典接口

- 日志注解

- 基础异常类

- 树形数据结构

- NewType 接口（参考 Rust 的 NewType 机制）

### tao-spring-boot-starter
